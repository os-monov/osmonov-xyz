### Welcome

Hey, my name is Alex. This repository holds the code for my personal website.


## Setup

Check your node version using:

```bash
node --version
```

If needed, install & use the needed version using nvm:
```bash
nvm install 18.15
nvm use 18.15
```

## Server Commands

To run the development server:

```bash
npm run dev
```


## Deploy

Generate a production build:

```bash
npm run build
```

Upload .out/ to the 'osmonov-xyz-site' S3 Bucket.
